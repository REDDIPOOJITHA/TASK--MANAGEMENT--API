# Task Management REST API

A backend RESTful API for managing tasks, designed with clean architecture and scalability in mind.

## Features
- Create and retrieve tasks
- RESTful API design
- Layered backend architecture

## Tech Stack
- Python, Flask
- SQLAlchemy
- REST APIs

## API Endpoints
- GET /tasks
- POST /tasks

## How to Run
```bash
pip install -r requirements.txt
python src/app.py
